## Run Locally

Clone the project

```bash
  git clone https://github.com/sheikhDipta003/PythonProjects.git
```

Go to the project directory

```bash
  cd PythonProjects
```

Install packages

```bash
  pip install -r requirements.txt
```

Run the python application

```bash
  python <file_name>  # e.g. main.py
```
